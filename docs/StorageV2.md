# Solidity API

## StorageV2

This smart contract is used to store numbers.

### number

```solidity
uint256 number
```

Stored number

### store

```solidity
function store(uint256 num) public
```

Update the stored number

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| num | uint256 | new number to store |
